Select2可以直接追加bootstrap风格的inputgroup组件（ Bootstrap Input Group）, 通过添加 .select2-bootstrap-prepend 或者 .select2-bootstrap-append 到 .input-group 控件中，来达到各种样式效果。
Bootstrap Control Sizing

组件提供了 small 和 large 两种大小的Select2样式 Bootstrap Input Groups 和 Bootstrap Control Sizing的样式可以通过(e. g. Select2 in .input-group.input-group-sm 或者 .input-group.input-group-lg). 你可以应用 Bootstrap Control Sizing 的class直接设置 .select2-container 来修改它的外形。
